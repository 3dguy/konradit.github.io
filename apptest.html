HeroPro App

100% unofficial open-source HTML5 GoPro app.

Includes live preview, full control of the camera, media browsing...

HeroPro App is FREE and OPEN SOURCE!!!

DOWNLOAD

You can download the index file for smartphones and computers in the embedded repo below

You can also use the URLs:

  • Mobile: http://m.heropro.chernowii.com
  • PC: http://pc.heropro.chernowii.com

The URLs have been tested with Mac OS X, Windows 7/8, Linux GNU, iOS, Android, Windows Phone 8, Blackberry OS

Native?

Yes, the app has been ported to native via PhoneGap, they are also in the repo.
